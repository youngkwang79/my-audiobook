# Project Custom Rules

## ✍️ 블로그 통합 포스팅 생성 및 업로드 규칙 (Platform Multi-Poster Rule)

유저가 새로운 원고 또는 키워드로 글 업로드 패키지를 요청할 경우, 반드시 다음 상세 기준을 만족하여 수행한다.

1. **포커스 키워드**:
   * AI가 제공된 문서나 소스를 분석하여 가장 경쟁력 있고 구글 검색 유입량이 높은 포커스 키워드를 스스로 추출하여 모든 포스팅 작업에 자동으로 적용한다.

2. **📚 무림북용**:
   * **이미지**: 2:3 비율 북커버 썸네일 생성 시, 텍스트(제목)가 이미지 중심부에 디자인으로 직접 합성되어 박혀 있는 최종 결과물을 사용한다.
   * **예약 발행**: 업로드 시점 기준 정확히 1시간 뒤 자동 예약 발행 설정 (`status = "공개예정"`, `created_at = 현재+1시간`).
   * **SEO 보강**: 본문 최하단에 자연스러운 내부 링크(부자 습관 칼럼 등), 외부 신뢰 링크(위키백과 등) 및 해시태그 강제 주입.
   * **텍스트 강조**: 독자 가독성 및 스캔율 향상을 위해 본문의 핵심 요지 및 핵심 구절은 굵게(Bold) 강조 처리한다.

3. **🔵 워드프레스용**:
   * **Rank Math SEO 90점 이상 기준 충족**:
     * 글자 수 1,000자 이상으로 문맥 확장.
     * 키워드 밀도 보정 (본문 내 포커스 키워드 적절히 분산 배치).
     * 내부 링크(부자 습관 칼럼) 및 외부 신뢰 링크 포함.
     * 본문 이미지에 Alt 대체 텍스트(Alt Text) 설정 필수.
     * 가독성 향상을 위해 본문의 핵심 문장은 굵게(Bold) 강조 처리한다.
   * **메타 설정**:
     * 글 메뉴에 관련 태그 추가.
     * 카테고리 3개 체크표시 선택 (`AI/자동화`, `IT/업무`, `경제/산업` 등).
     * 제목 규칙: 핵심키워드를 맨 앞에 배치하고, 숫자를 포함하여 공백 포함 20자 내외로 짓는다.
   * **미디어 연동**:
     * 16:9 규격의 대표 이미지(Featured)와 16:9 본문 중간 이미지 생성 시, 텍스트가 직접 디자인 인쇄되어 있는 최종 이미지들을 사용하고 이를 워드프레스 미디어 서버에 자동 업로드하고 스니펫 메타데이터 설정하여 임시글(Draft)로 포스팅한다.

4. **🟢 구글 블로거용 (Blogger API)**:
   * 중복 문서 페널티(저품질 블로그 낙인)를 피하기 위해, 동일한 주제를 완전히 다른 시각(정보 전달/테크니컬 소비전력 분석 위주)의 원고로 변환하여 임시 저장글로 자동 업로드한다.
   * **이미지 전송 규칙**: 본문 맨 상단에 텍스트가 인쇄된 16:9 또는 2:3 규격의 썸네일 대표 이미지를 포함해 전송하며, **이미지 하단에 해당 이미지의 맥락을 가독성 있게 표현한 친절한 설명문(Caption) 태그를 항상 필수로 동봉**한다.
   * **가독성 보강**: 글 내부의 핵심 사실과 키워드가 포함된 문맥은 굵게(Bold) 찐하게 표기하여 전송한다.

5. **📂 티스토리 & 네이버 블로그용**:
   * **티스토리**: 마크다운 서식 전용 카카오/스마트홈 타깃 원고 (핵심 강조 반영).
   * **네이버**: 줄글이 짧고 이모티콘과 공감이 풍부한 친근한 에세이형 모바일 가독성 원고 (핵심 강조를 한국어 굵게 반영하고 특수 기호 100% 제거).
